Node.js example.

```bash
mkdir test1 && cd test1
cat <<EOF > index.js
module.exports = (req, res) => {
  res.end('Hello from Node.js on Now 2.0!');
};
EOF
cat <<EOF > now.json
{
    "version": 2,
    "name": "nodejs",
    "builds": [
        { "src": "*.js", "use": "@now/node" }
    ]
}
EOF
now
```

React.js example.

```bash
npx create-react-app test1 && cd test1
cat <<EOF > now.json
{
    "version": 2,
    "name": "test1",
    "builds": [
        { "src": "package.json", "use": "@now/static-build" }
    ],
    "routes": [
        {"src": "^/static/(.*)", "dest": "/static/$1"},
        {"src": ".*", "dest": "/index.html"}
    ]
}
EOF
now
```
