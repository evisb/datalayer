## Build

```bash
cd $DLAHOME/repos/nteract && \
  yarn && \
  yarn build:packages
```

```bash
# Build and watch packages.
yarn build:packages:watch
# In another shell.
cd packages/kernel-relay && \
  yarn dev
```

```bash
# Build packages separately.
tsc -b packages/ansi-to-react
yarn build:only @nteract/actions
```
