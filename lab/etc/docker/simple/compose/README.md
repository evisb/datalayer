# Docker Simple Compose

Example from https://docs.docker.com/compose/gettingstarted

```bash
# Optional: Build.
docker build -t datalayer/simple-compose:0.0.1 .
```

```bash
# Start.
echo http://0.0.0.0:5000 && \
  docker-compose up -d && \
  docker-compose logs -f
```

```bash
# Stop.
docker-compose down
```
